# Bemer

Build reusable UI components for Rails applications using the [BEM methodology](https://en.bem.info/methodology/).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bemer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bemer
